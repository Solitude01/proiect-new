### 使用流程

1. **添加新任务**：点击"添加新的处理任务"
2. **选择文件**：点击"点击选择 PPT 文件"，选择文件后该区域自动隐藏
3. **开始处理**：点击"开始处理"按钮
4. **等待完成**：查看进度条，等待处理完成
5. **下载文件**：
   - 方式一：点击"展开"按钮，在弹出的模态框中下载
   - 方式二：直接点击"下载 JSON"或"下载演讲稿"按钮
6. **重新使用**：下载演讲稿后，uploadArea 重新显示，可以添加新文件

### 技术要点

- 使用 Map 存储卡片数据，支持多任务并行处理
- 使用 setTimeout 清理下载相关的 URL 对象，避免内存泄漏
- 通过 display 属性控制 uploadArea 的显示/隐藏
- 完整的错误处理和用户反馈机制
