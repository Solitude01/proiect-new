在你的 Docker 环境里，社区节点本质上就是 npm 包。安装步骤和 n8n-nodes-powerpoint2text 类似：

步骤 A：Dockerfile 里安装

在你的 Dockerfile 中添加一行：

        # 安装额外社区节点，例如 n8n-nodes-google
        RUN npm install --omit=dev n8n-nodes-google --prefix /home/node/.n8n


✅ 注意：

--prefix /home/node/.n8n 保证节点安装到 n8n 的自定义节点目录。

安装完成后记得 chown -R node:node /home/node/.n8n，保证 n8n 用户有权限。

        步骤 B：容器内直接安装（临时）
            docker exec -it n8n-python sh
            cd /home/node/.n8n
            npm install --omit=dev n8n-nodes-google


这样安装后，重启容器节点即可在 n8n UI 里看到新节点，但此方法重建镜像会丢失。

步骤 C：确保 n8n 识别

N8N_CUSTOM_NODES_PATH=/home/node/.n8n 已在环境变量里设置。

N8N_ALLOW_LOADING_UNSIGNED_NODES=true 确保未签名节点也能加载。
