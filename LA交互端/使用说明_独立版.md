# LA交互端 - 独立运行版

## 运行要求
- Windows 10/11 或 Windows Server 2016+
- 无需安装 Python
- 内存: 至少 2GB
- 磁盘: 至少 50MB 可用空间

## 启动方式
1. 将整个文件夹复制到目标电脑
2. 双击运行 `start.bat`（纯英文文件名，避免编码问题）
3. 使用固定端口启动：
   - 管理控制台：8001
   - 业务视图：6010
4. 根据提示的地址访问相应服务

**注意**：使用纯英文文件名（LAdmin.exe / LView.exe / start.bat）以避免目标电脑编码问题

**端口说明**：当前版本使用固定端口（8001和6010），如需修改请编辑 `start.bat` 文件修改 `MAIN_PORT` 和 `VIEW_PORT` 的值

## 功能特点
- **固定端口配置**: 使用固定端口 8001（管理控制台）和 6010（业务视图）
- **双服务架构**: 同时启动管理控制台和业务视图服务
- **纯英文文件名**: 避免目标电脑编码问题
- **自动错误处理**: 启动失败时会显示错误信息

## 访问地址
启动成功后使用固定端口访问：

**管理控制台**（主服务）：
- http://localhost:8001

**业务视图**（只读访问）：
- http://localhost:6010

如需修改端口，请编辑 `start.bat` 文件中的 `MAIN_PORT` 和 `VIEW_PORT` 变量。

## 注意事项
1. 首次运行请允许 Windows 防火墙访问
2. 确保以下端口未被其他程序占用：
   - 8001（管理控制台服务）
   - 6010（业务视图服务）
3. 如需修改端口，请编辑 `start.bat` 文件
4. 配置文件保存在 `configs/instances/` 目录
5. 日志文件保存在 `logs/` 目录

## 常见问题

### Q: 端口被占用怎么办？
A: 如果 8001 或 6010 端口被占用，请编辑 `start.bat` 文件，修改 `MAIN_PORT` 或 `VIEW_PORT` 的值为其他可用端口。

### Q: 为什么有两个服务？
A: 系统采用双服务架构：
- **管理控制台（端口8000+）**：提供完整的实例管理、配置编辑、控制面板功能
- **业务视图（端口6010+）**：提供只读的业务数据展示，适合车间大屏展示

### Q: 只启动了一个服务怎么办？
A: 如果某个端口范围的端口都被占用，对应的服务将不会启动。主服务（管理控制台）是必须的，业务视图是可选的。

### Q: 如何停止服务?
A: 关闭启动脚本窗口即可停止所有服务，或按 `Ctrl+C` 停止。

### Q: 如何修改端口?
A: 编辑 `start.bat` 文件，修改以下变量的值：
- `MAIN_PORT`：管理控制台端口（默认 8001）
- `VIEW_PORT`：业务视图端口（默认 6010）
