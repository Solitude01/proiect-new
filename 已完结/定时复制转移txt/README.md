# TXT同步工具 v3.0

一个基于Python的TXT文件自动同步工具，支持系统托盘运行，可定时将源文件夹中的TXT文件同步到目标文件夹。

## 功能特点

- **批量同步**: 自动扫描源文件夹（含子目录）中的所有TXT文件
- **智能更新**: 比较文件修改时间，仅同步新增或更新的文件
- **系统托盘**: 支持最小化到系统托盘后台运行
- **定时轮询**: 可配置轮询间隔，自动检测文件变化
- **扁平复制**: 所有文件复制到目标文件夹根目录，不保留目录结构
- **配置持久化**: 支持保存和加载配置文件

## 环境要求

- Python 3.6+
- Windows 操作系统

## 依赖安装

```bash
pip install pystray pillow
```

## 使用方法

### 启动程序

```bash
python TXT同步工具_v3.0.py
```

### 界面配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 源文件夹路径 | 需要同步的TXT文件所在目录 | `D:\Documents\Source` |
| 目标文件夹路径 | 同步目标目录 | `D:\Documents\Target` |
| 轮询间隔(秒) | 检查文件变化的时间间隔 | `5` |
| 文件扩展名 | 需要同步的文件类型 | `.txt` |
| 排除文件 | 需要排除的文件名关键词 | `temp,cache,.tmp` |

### 操作按钮

- **开始同步**: 启动定时同步任务
- **停止同步**: 停止当前同步任务
- **保存配置**: 保存当前配置到 `sync_config.ini`
- **最小化到托盘**: 隐藏窗口到系统托盘

### 窗口行为

- 点击**最小化按钮**: 最小化到任务栏
- 点击**关闭按钮(X)**: 最小化到系统托盘（不退出程序）
- 退出程序: 仅能通过系统托盘右键菜单退出

## 同步日志说明

| 日志标识 | 含义 |
|----------|------|
| `📄 新增: xxx.txt` | 新文件已同步 |
| `🔄 更新: xxx.txt` | 已更新的文件已覆盖同步 |
| `ℹ️ 当前没有新文件或更新需要同步` | 无变化 |
| `❌ 文件复制失败` | 同步出错 |

## 配置文件

程序会在同目录下生成 `sync_config.ini` 配置文件：

```ini
[PATHS]
SHARED_FOLDER_PATH = D:\Source
LOCAL_TARGET_PATH = D:\Target
LOG_FILE_PATH = sync_log.log

[SYNC]
POLLING_INTERVAL_SECONDS = 5
RECURSIVE_SYNC = True
REMOVE_EXTRA_FILES = false

[FILTER]
FILE_EXTENSIONS = .txt
EXCLUDE_ITEMS = temp,cache,.tmp,Thumbs.db,desktop.ini

[TRAY]
SHOW_TRAY_ICON = True
MINIMIZE_TO_TRAY = True
```

## 系统托盘菜单

右键点击系统托盘图标可访问：

- 打开主界面
- 开始同步
- 停止同步
- 显示状态
- 保存配置
- 退出程序

## 注意事项

1. 首次运行需配置源文件夹和目标文件夹路径
2. 同名文件会根据修改时间判断是否需要更新
3. 目标文件夹不存在时会自动创建
4. 建议将轮询间隔设置为5秒以上，避免频繁IO操作
5. 如未安装 `pystray` 库，程序仍可运行但无系统托盘功能
