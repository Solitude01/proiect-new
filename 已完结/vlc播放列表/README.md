# VLC播放列表转换器 - GUI版本使用说明

## 功能介绍

这个GUI版本的VLC播放列表转换器支持以下功能：

1. **支持多种文件格式**：
   - Markdown (.md) 文件
   - Excel (.xlsx/.xls) 文件
   - 文本 (.txt) 文件

2. **智能格式处理**：
   - 自动清理多余空格
   - 自动删除换行符
   - 验证URL格式

3. **Excel文件特殊支持**：
   - 可自定义监控点名称列（默认A列）
   - 可自定义流地址列（默认B列）

## 安装依赖

首次使用前，请安装所需依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动程序

```bash
python vlc播放列表转换器_GUI.py
```

### 2. 基本操作

#### 处理Markdown文件
1. 点击"浏览..."选择Markdown输入文件
2. 选择输出文件路径（会自动生成默认路径）
3. 确保"Markdown (.md)"选项被选中
4. 根据需要调整格式清理选项
5. 点击"开始转换"

#### 处理Excel文件
1. 点击"浏览..."选择Excel输入文件（支持.xlsx和.xls格式）
2. 选择输出文件路径
3. 选择"Excel (.xlsx/.xls)"选项
4. 在弹出的配置区域设置：
   - 监控点名称列（默认A列）
   - 流地址列（默认B列）
5. 根据需要调整格式清理选项
6. 点击"开始转换"

### 3. 格式要求

#### Markdown文件格式
每行应该包含监控点名称和RTSP流地址，用空格分隔：
```
监控点1名称 rtsp://username:password@ip:port/path
监控点2名称 rtsp://username:password@ip:port/path
```

#### Excel文件格式
- A列：监控点名称
- B列：RTSP流地址
- 支持.xlsx和.xls格式
- 支持自定义列配置

### 4. 输出格式

程序会生成标准的M3U播放列表格式，包含EXTM3U文件头：
```
#EXTM3U
#EXTINF:-1,监控点1名称
rtsp://username:password@ip
#EXTINF:-1,监控点2名称
rtsp://username:password@ip
```

## 特殊功能

### 格式清理
- **删除多余空格**：去除首尾空格和连续多个空格
- **删除换行符**：清理文本中的换行符和回车符

### URL处理
- 自动解析RTSP URL
- 去除端口号（保留用户名和密码）
- 验证URL格式

### 错误处理
- 详细的处理日志
- 错误行的跳过和报告
- 完整的进度显示

## 命令行版本对比

与原命令行版本相比，GUI版本的优势：

1. **用户友好**：可视化界面，无需修改代码
2. **多格式支持**：同时支持Markdown和Excel
3. **灵活配置**：可自定义Excel列配置
4. **实时反馈**：进度条和详细日志
5. **错误处理**：更好的错误提示和处理

## 故障排除

### 常见问题

1. **Excel文件打不开**
   - 确保安装了openpyxl库
   - 检查Excel文件是否损坏

2. **处理失败**
   - 检查输入文件路径是否正确
   - 确认文件格式是否符合要求
   - 查看错误日志获取详细信息

3. **依赖问题**
   - 确保Python版本>=3.6
   - 安装所有必需的依赖包
