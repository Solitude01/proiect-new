# 坤云清理管理工具 - 使用说明

## 部署

将以下 4 个文件放在**同一目录**：

```
├── 坤云清理管理工具.exe    # GUI 管理工具
├── config.json             # 配置文件
├── cleanup.ps1             # 清理脚本
└── install.ps1             # 计划任务安装脚本
```

双击 `坤云清理管理工具.exe` 即可运行，无需安装 Python 或其他依赖。

---

## 四个功能标签页

### 1. 配置编辑

编辑 `config.json` 的可视化界面。

- **平台名称**：标识当前部署的平台
- **清理任务列表**：查看/添加/编辑/删除清理任务
  - 编辑时可从下拉框快速选择文件类型（图片类型 / 视频类型 / 日志类型 / 自定义）
  - 两种清理模式：
    - **bySize**（按大小）：目录超过阈值时，从最早的文件开始删除，直到低于阈值
    - **byDays**（按天数）：删除超过指定天数的文件
- **全局设置**：日志目录、日志保留天数、模拟运行开关
- 修改后点击「保存配置」写入 config.json

> **注意**：请确认文件类型与目录中的实际文件匹配，否则脚本找不到匹配文件，不会执行清理。

### 2. 手动执行

立即运行一次 cleanup.ps1，实时显示日志输出。

- 点击「执行清理」开始，日志按级别着色（红=错误 / 黄=警告 / 绿=成功）
- 可随时点击「停止」终止执行
- 支持「导出」日志到文件

### 3. 模拟运行

以 dryRun 模式执行，**不会实际删除任何文件**，仅显示将要执行的操作。

- 用于验证配置是否正确、预估将删除的文件数和释放的空间
- 运行结束后顶部显示汇总信息

### 4. 计划任务

管理 Windows 计划任务，实现自动定时清理。

- **查看状态**：任务名称、状态（Ready/Running/NotFound）、上次运行时间和结果、下次运行时间
- **安装计划任务**：设置触发间隔（1-24 小时），点击安装，弹出 UAC 提权确认
- **卸载计划任务**：移除已注册的计划任务
- **立即触发一次**：手动触发计划任务执行（不经过 GUI，直接由系统调度器执行）
- **刷新状态**：重新查询计划任务的最新状态

> 安装/卸载/触发/停止操作均需要管理员权限，会弹出 UAC 提示。

---

## 计划任务说明

计划任务安装后：

- **不需要**保持 GUI 软件运行，任务由 Windows 系统调度器自动管理
- 以 SYSTEM 账户运行，具备最高权限
- 开机自动启动 + 按设定间隔重复执行
- 执行日志写入 config.json 中配置的日志目录（默认每次运行生成独立的 `cleanup_YYYYMMDD_HHmmss.log`）
- 旧日志文件按配置的保留天数自动清理

如需修改清理配置，打开 GUI 修改 config.json 并保存即可，**不需要重新安装计划任务**（计划任务每次执行时读取最新的 config.json）。

如需修改执行间隔，需重新安装计划任务（会自动覆盖旧任务）。

## 常见问题

**Q: 任务安装后状态显示 NotFound？**
A: 确认 install.ps1、cleanup.ps1、config.json 与 exe 在同一目录。安装时请在弹出的 PowerShell 窗口中观察是否有错误信息。

**Q: 手动执行后文件没有被清理？**
A: 检查配置的文件类型是否与目录中的实际文件扩展名匹配。例如监控图片目录应选择「图片类型」而非「视频类型」。可先用「模拟运行」验证。

**Q: 日志显示"目录大小未超过阈值，无需清理"？**
A: 说明当前目录大小在设定范围内，属于正常情况。

**Q: 如何查看计划任务的执行日志？**
A: 到 config.json 中配置的 `logDir` 目录下查看 `cleanup_*.log` 文件。
