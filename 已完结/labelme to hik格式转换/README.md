# Labelme标注转换工具

将Labelme标注格式转换为Hikvision设备支持的标注格式的工具。支持单检测模式和混合标注模式两种转换方式。

## 📋 功能特性

### 🎯 核心功能
- ✅ **单检测模式**: 简单的单级检测标注转换
- ✅ **混合标注模式**: 支持多级分类的复杂标注转换
- ✅ **批量处理**: 支持批量转换多个标注文件
- ✅ **智能标签映射**: 自动识别和配置标签分类
- ✅ **进度监控**: 实时显示转换进度和状态

### 🛠️ 工具功能
- 📊 **标签扫描**: 自动扫描输入文件夹中的所有标签
- 🔧 **标签配置**: 灵活的标签映射和分类配置
- 💾 **配置导入/导出**: 支持保存和加载配置
- 🤖 **智能推荐**: 基于标签内容的智能分类推荐
- 📁 **文件管理**: 自动复制图片文件到输出目录

## 🚀 快速开始

### 环境要求
- Python 3.6+
- tkinter (Python标准库)
- json, os, glob, shutil等标准库

### 安装和运行

#### 方法1: 使用GUI程序（推荐）
```bash

```

#### 方法2: 使用命令行程序

## 📖 使用说明

### GUI界面使用

1. **选择文件夹**
   - 输入文件夹: 包含Labelme JSON标注文件和对应图片的文件夹
   - 输出文件夹: 转换后文件的保存位置

2. **选择转换模式**
   - **单检测模式**: 适用于简单的单级检测任务
   - **混合标注模式**: 适用于需要多级分类的复杂标注任务

3. **配置标签（混合模式）**
   - 点击"扫描标签"自动识别所有标签
   - 配置检测标签名、一级分类、二级分类
   - 支持批量配置和智能推荐

4. **开始转换**
   - 点击"开始转换"按钮
   - 查看实时进度和日志信息
   - 转换完成后查看输出文件

### 命令行使用

```bash
python 标注转换0.3.py
```

按提示输入输入文件夹和输出文件夹路径即可。

## 🎛️ 转换模式详解

### 单检测模式
- **适用场景**: 简单的物体检测任务
- **特点**: 每个标签直接对应检测目标
- **输出**: 简化的标注格式，适合基础检测需求

### 混合标注模式
- **适用场景**: 复杂的分类检测任务
- **特点**: 支持多级分类体系
- **配置项**:
  - **检测标签名**: 用于识别的具体标签名
  - **一级分类**: 标签的大类分组
  - **二级分类**: 标签的细分分类

## 📊 标签配置

### 配置字段说明
- **原标签名**: Labelme文件中的原始标签名称
- **使用次数**: 该标签在所有标注文件中出现的次数
- **检测标签名**: 用于检测的标签名称（可自定义）
- **一级分类**: 标签的顶级分类
- **二级分类**: 标签的细分分类
- **配置状态**: 显示配置完成度（未配置/部分配置/完全配置）

### 配置方法
1. **手动配置**: 双击表格单元格直接编辑
2. **快速配置**: 选中标签后在下方输入框配置
3. **批量配置**: 使用"批量应用常用分类"功能
4. **智能推荐**: 使用"智能推荐分类"自动推荐

## 🐛 常见问题

### Q: 转换失败，提示找不到图片文件
A: 确保输入文件夹中包含Labelme JSON文件和对应的图片文件，图片格式支持JPG、PNG、BMP等。

### Q: 标签配置状态显示"未配置"
A: 在混合标注模式下，需要为每个标签配置检测标签名、一级分类和二级分类。

### Q: 转换速度慢
A: 可以先使用"扫描标签"功能，只对有标注的图片进行转换，避免处理空文件。

### Q: 如何批量配置相似标签
A: 使用"批量应用常用分类"功能，选择预设的分类模板批量应用到相似标签。
