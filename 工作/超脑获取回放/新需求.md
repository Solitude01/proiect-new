# 需求文档

## 一、辅助监控系统（新增需求）
### 辅助监控 —— LA 预警对接

1. **预警对接方式**
   - 通过 HTTP 请求发送预警信息。
   - 预警内容包含以下字段（字段需进一步确认）：
     - 唯一ID
     - 预警时间
     - 开始时刻
     - 结束时刻
     - 超脑
     - 通道号
     - Task Result


2. **项目管理与任务点检**
   - 在项目管理中为每个项目配置唯一 ID。
   - 平台基于http请求 向超脑获取准确的视频数据。
   - 视频结果需在任务中展示，用于点检。

3. **系统对接与协作**
   - 查看：D:\proiect\工作\超脑获取回放 

4. **时间一致性校验**
   - 需要对齐超脑数据集成处理平台服务器的系统时钟-同意对齐南通时钟源--10.30.5.100 ：123
   - 以保证预警时间、视频时间的一致性和准确性。
