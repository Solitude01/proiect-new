  1. build.bat 修复内容

  在 pip install 之前清除了 http_proxy / https_proxy 环境变量。这是之前打包失败的原因 — 公司/VPN 代理地址格式有误导致 pip 无法解析 URL。

  ---
  2. 直接运行脚本（开发/调试用）

  前提： 已安装 Python 3.10+

  # 1) 进入项目目录
  cd D:\proiect\工作\LA\测试保存视频

  # 2) 创建虚拟环境（首次）
  python -m venv venv

  # 3) 激活虚拟环境
  venv\Scripts\activate

  # 4) 安装依赖（首次）
  pip install flask requests waitress

  # 5) 启动服务
  python app.py

  # 自定义端口/地址
  python app.py --host 0.0.0.0 --port 8080

  启动后浏览器访问 http://localhost:9800（默认端口）。

  ---
  3. 打包后使用指南

  打包： 双击 build.bat，产物在 dist/ 目录。

  部署： 将 dist/ 文件夹复制到目标机器，内含：

  ┌─────────────────┬────────────────────────────────────┐
  │      文件       │                说明                │
  ├─────────────────┼────────────────────────────────────┤
  │ NVR-Manager.exe │ 主程序（单文件，无需 Python 环境） │
  ├─────────────────┼────────────────────────────────────┤
  │ Deepmind.json   │ 设备/超脑通道映射配置              │
  └─────────────────┴────────────────────────────────────┘

  运行：

  # 默认启动（0.0.0.0:9800）
  NVR-Manager.exe

  # 指定端口
  NVR-Manager.exe --port 8080

  # 指定绑定地址和端口
  NVR-Manager.exe --host 127.0.0.1 --port 9900

  配置说明：

  首次启动会自动生成 config.json，包含以下可通过 Web 页面修改的参数：

  ┌──────────────────┬───────────────────────┬──────────────────────────────────────────────────┐
  │       参数       │        默认值         │                       说明                       │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ base_dir         │ ~/Downloads/NVR (Win) │ 视频保存目录                                     │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ pre_seconds      │ 5                     │ 事件前录像秒数                                   │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ post_seconds     │ 2                     │ 事件后录像秒数                                   │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ tail_seconds     │ 120                   │ 最大录像尾部秒数                                 │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ tracks_suffix    │ 01                    │ 录像通道后缀                                     │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ download_timeout │ 180                   │ 下载超时(秒)                                     │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ max_workers      │ 4                     │ 最大并行下载数                                   │
  ├──────────────────┼───────────────────────┼──────────────────────────────────────────────────┤
  │ download_delay   │ 20                    │ 新增 — webhook 到达后等待秒数，确保 NVR 录像完整 │
  └──────────────────┴───────────────────────┴──────────────────────────────────────────────────┘

  Webhook 地址： http://<服务器IP>:9800/webhook/event
