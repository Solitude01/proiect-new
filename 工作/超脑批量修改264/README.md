# 超脑设备编码格式批量修改工具

批量修改海康威视 NVR（超脑）设备监控点视频编码格式的 GUI 工具。

## 功能特点

- **自动发现通道**: 自动扫描每台超脑设备的所有监控点通道
- **显示监控点名称**: 扫描后显示每个通道的监控点名称（摄像头名称）
- **编码检查**: 获取每个通道的当前编码格式（H.264/H.265）
- **搜索功能**: 支持按监控点名称、通道ID、IP地址搜索过滤
- **导出监控点**: 一键导出所有监控点信息到 CSV 文件
- **灵活选择**: 支持全选、单选、按设备选择等多种选择方式
- **单个设备扫描**: 支持在顶部工具栏选择单个设备进行扫描
- **批量修改**: 一键将选中的 H.265 通道批量修改为 H.264
- **实时日志**: 显示详细的操作日志和进度

## 环境要求

- Python 3.7+
- requests 库

## 安装

```bash
# 安装依赖
pip install -r requirements.txt
```

## 使用方法

### 1. 配置文件

确保当前目录有 `Deepmind.json` 文件，格式如下：

```json
[
    {
        "Deepmind": "10",
        "IP": "10.30.5.186",
        "Password": "wdzn6688",
        "HttpPort": 80,
        "RtspPort": 554,
        "Scheme": "http"
    }
]
```

### 2. 运行程序

```bash
python nvr_codec_manager.py
```

### 3. 操作步骤

1. **加载设备**: 程序启动时自动加载 `Deepmind.json` 中的设备列表
2. **扫描设备**: 
   - 点击「🔍 扫描所有设备」获取所有设备的通道和编码信息
   - 或在顶部下拉框选择设备后点击「🔍 扫描该设备」扫描单个设备
3. **搜索监控点**: 在搜索框输入监控点名称、通道ID或IP地址进行过滤
4. **选择通道**:
   - 勾选需要修改的通道（建议只选 H.265 的通道）
   - 或点击「✓ 全选H.265」自动选中所有 H.265 通道
5. **导出监控点**: 点击「📥 导出监控点」将所有监控点信息导出为 CSV 文件
6. **执行修改**: 点击「▶ 执行修改」开始批量修改

## 界面说明

### 设备卡片

每个设备显示为一张卡片，包含：
- **设备信息**: IP 地址和 Deepmind ID
- **连接状态**: 已连接/未连接/连接失败
- **通道列表**: 每个通道显示监控点名称和编码格式（✓ H.264 / ✗ H.265）
- **操作按钮**: 全选本设备、取消全选、仅扫描本设备

### 工具栏

| 按钮 | 功能 |
|------|------|
| 🔄 刷新设备 | 重新加载 Deepmind.json |
| 🔍 扫描所有设备 | 扫描所有设备的通道和编码 |
| 扫描该设备 | 扫描顶部下拉框中选中的单个设备 |
| 搜索框 | 按监控点名称/通道ID/IP地址搜索过滤 |
| 📥 导出监控点 | 导出所有监控点信息到 CSV 文件 |
| ✓ 全选H.265 | 自动选中所有 H.265 通道 |
| ✗ 取消全选 | 取消所有选择 |

### 底部状态栏

- **已选统计**: 显示当前选中的通道数量
- **进度条**: 显示扫描/修改进度
- **执行修改**: 开始批量修改

## 导出文件格式

导出的 CSV 文件包含以下字段：

| 字段 | 说明 |
|------|------|
| 序号 | 监控点序号 |
| 设备IP | 超脑设备IP地址 |
| Deepmind ID | 设备ID |
| 通道ID | 通道编号 |
| 监控点名称 | 摄像头名称 |
| 码流ID | 码流标识 |
| 当前编码 | H.264 / H.265 / 未知 |
| 状态 | 设备连接状态 |

## 注意事项

1. **网络要求**: 运行此工具的电脑需要能访问超脑设备的 IP 地址
2. **账号权限**: 使用 admin 账号和密码进行 Digest 认证
3. **编码修改影响**:
   - 修改编码格式会导致视频流短暂中断（1-3秒）
   - H.264 比 H.265 占用更多存储空间和带宽
4. **先扫描后修改**: 必须先扫描获取通道信息，才能执行修改

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 连接超时 | 检查网络连接和设备 IP 是否可达 |
| 获取配置失败 | 确认密码正确，设备 ISAPI 接口正常 |
| 修改失败 | 检查设备是否在线，通道是否存在 |
| 监控点名称显示"未知" | 设备可能未配置通道名称，或接口返回异常 |

## 技术原理

通过海康威视 ISAPI 接口操作：
1. `GET /ISAPI/ContentMgmt/InputProxy/channels` - 获取通道列表（包含监控点名称）
2. `GET /ISAPI/ContentMgmt/StreamingProxy/channels/{id}` - 获取通道配置
3. `PUT /ISAPI/ContentMgmt/StreamingProxy/channels/{id}` - 修改编码格式

认证方式: HTTP Digest Authentication
