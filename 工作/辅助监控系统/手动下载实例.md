     1. HTTP 下载 URL（浏览器/curl）

     http://10.30.5.112:80/ISAPI/ContentMgmt/download?playbackURI=rtsp://10.30.5.112:554/Streaming/tracks/101?starttime=20260302T160620Z%26endtime=20260302T160640Z

     注意：& 必须编码为 %26，否则会被当作 HTTP query 参数分隔符。

     认证方式：HTTP Digest Auth，用户名 admin，密码 wdzn6688

     2. curl 命令

     curl --digest -u admin:wdzn6688 \
       "http://10.30.5.112:80/ISAPI/ContentMgmt/download?playbackURI=rtsp://10.30.5.112:554/Streaming/tracks/101?starttime=20260302T160620Z%26endtime=20260302T160640Z" \
       -o ch1_20260302T160620Z_20260302T160640Z.mp4

     3. Python 脚本命令

     python download_event.py \
       --event_id "manual_test" \
       --beginTime "2026-03-02 16:06:20.464" \
       --endTime "2026-03-02 16:06:40.464" \
       --Deepmind "20" \
       --Channel "1" \
       --base_dir "." \
       --deepmind_json "Deepmind.json"

     4. RTSP 实时回放 URL（VLC/PotPlayer）

     rtsp://admin:wdzn6688@10.30.5.112:554/Streaming/tracks/101?starttime=20260302T160620Z&endtime=20260302T160640Z

     RTSP 地址中 & 不需要编码。
