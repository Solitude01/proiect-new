# 海康NVR回放接口文档

> NVR型号: 海康威视
> 认证方式: HTTP Digest (MD5)
> 测试设备: `10.30.5.112:80`

---

## 一、下载回放视频（MP4文件）

### 基本信息

| 项目 | 值 |
|------|-----|
| 接口路径 | `/ISAPI/ContentMgmt/download` |
| 请求方式 | `GET` |
| 认证方式 | HTTP Digest Auth |
| 返回类型 | `Opaque/data`（MP4视频流） |

### 请求参数

通过 URL Query 传参：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| playbackURI | String | 是 | RTSP回放地址，包含通道、起止时间 |

`playbackURI` 格式：

```
rtsp://{NVR_IP}:{NVR_PORT}/Streaming/tracks/{通道号}01?starttime={开始时间}&endtime={结束时间}
```

> 注意：playbackURI 中的 `&` 需要编码为 `%26`

### 时间格式

```
YYYYMMDDTHHmmSSZ
```

示例：`2026-02-10 13:32:05` → `20260210T133205Z`

### 完整请求示例

```
GET http://10.30.5.112:80/ISAPI/ContentMgmt/download?playbackURI=rtsp://10.30.5.112:80/Streaming/tracks/501?starttime=20260210T133205Z%26endtime=20260210T133516Z

Authorization: Digest username="admin", realm="...", nonce="...", uri="...", response="..."
```

### 返回说明

| 状态码 | Content-Type | 说明 |
|--------|--------------|------|
| 200 | Opaque/data | 成功，返回视频二进制流，可直接保存为 `.mp4` |
| 401 | text/xml | 认证失败 |

### Python 调用示例

```python
import requests
from requests.auth import HTTPDigestAuth

url = (
    "http://10.30.5.112:80/ISAPI/ContentMgmt/download?"
    "playbackURI=rtsp://10.30.5.112:80/Streaming/tracks/501"
    "?starttime=20260210T133205Z%26endtime=20260210T133516Z"
)

resp = requests.get(
    url,
    auth=HTTPDigestAuth("admin", "wdzn6688"),
    stream=True,
    timeout=30
)

if resp.status_code == 200:
    with open("playback.mp4", "wb") as f:
        for chunk in resp.iter_content(chunk_size=8192):
            if chunk:
                f.write(chunk)
```

## 二、RTSP回放链接（在线播放）

### 基本信息

| 项目 | 值 |
|------|-----|
| 协议 | RTSP |
| 默认端口 | `554` |
| 认证方式 | URL内嵌用户名密码 |
| 播放工具 | VLC、ffplay、PotPlayer 等支持RTSP的播放器 |

### URL格式

```
rtsp://{用户名}:{密码}@{NVR_IP}:{RTSP端口}/Streaming/tracks/{通道号}01?starttime={开始时间}&endtime={结束时间}
```

### 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户名 | String | 是 | NVR登录用户名 |
| 密码 | String | 是 | NVR登录密码 |
| NVR_IP | String | 是 | NVR设备IP地址 |
| RTSP端口 | Number | 是 | RTSP服务端口，默认 `554` |
| 通道号 | String | 是 | 视频通道号（如 `5`，拼接后为 `501`） |
| starttime | String | 是 | 开始时间，格式 `YYYYMMDDTHHmmSSZ` |
| endtime | String | 是 | 结束时间，格式 `YYYYMMDDTHHmmSSZ` |

### 通道号规则

```
通道号 + "01" = trackID

通道1  → 101
通道5  → 501
通道11 → 1101
```

`01` 表示主码流，`02` 表示子码流。

### 完整请求示例

```
rtsp://admin:wdzn6688@10.30.5.112:554/Streaming/tracks/501?starttime=20260210T133205Z&endtime=20260210T133516Z
```

### 使用方式

**VLC 命令行播放：**

```bash
vlc "rtsp://admin:wdzn6688@10.30.5.112:554/Streaming/tracks/501?starttime=20260210T133205Z&endtime=20260210T133516Z"
```

**ffplay 播放：**

```bash
ffplay -rtsp_transport tcp "rtsp://admin:wdzn6688@10.30.5.112:554/Streaming/tracks/501?starttime=20260210T133205Z&endtime=20260210T133516Z"
```

**Python 生成链接：**

```python
from datetime import datetime

def get_rtsp_url(nvr_ip, rtsp_port, username, password, channel, begin_time, end_time):
    """
    生成RTSP回放链接
    :param begin_time: "2026-02-10 13:32:05" 或 "2026-02-10 13:32:05.552"
    :param end_time:   同上
    :return: RTSP播放链接
    """
    dt1 = datetime.strptime(begin_time.split('.')[0], "%Y-%m-%d %H:%M:%S")
    dt2 = datetime.strptime(end_time.split('.')[0], "%Y-%m-%d %H:%M:%S")
    t1 = dt1.strftime("%Y%m%dT%H%M%SZ")
    t2 = dt2.strftime("%Y%m%dT%H%M%SZ")

    return (
        f"rtsp://{username}:{password}@{nvr_ip}:{rtsp_port}"
        f"/Streaming/tracks/{channel}01?starttime={t1}&endtime={t2}"
    )
```

## 附录：两种方式对比

| 对比项 | 下载视频 (HTTP) | RTSP链接 |
|--------|-----------------|----------|
| 协议 | HTTP GET | RTSP |
| 端口 | 80 | 554 |
| 认证 | Digest Auth | URL内嵌 |
| 输出 | MP4文件 | 实时流播放 |
| 浏览器支持 | 不支持（需Digest认证） | 不支持（需RTSP播放器） |
| 适用场景 | 存档、离线分析、二次处理 | 即时查看、快速回看 |
| 等待时间 | 需等下载完成 | 即时播放 |
| 占用磁盘 | 是 | 否 |
