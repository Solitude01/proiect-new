# 视频帧提取工具 v6.0

基于 FFmpeg 的视频帧提取桌面应用程序，支持单视频、单目录和多目录并发切分模式。

---

## 功能特性

- **三种切分模式**
  - 单视频切分 - 处理单个视频文件
  - 单目录切分 - 批量处理目录内所有视频
  - 多目录并发切分 - 同时处理多个目录，支持并发执行

- **灵活的切分设置**
  - 统一设置 - 所有视频使用相同的切分张数
  - 单独设置 - 为每个视频单独指定切分张数
  - 快捷按钮 - 50/100/150/200 一键设置

- **现代化UI**
  - ttkbootstrap flatly 主题
  - 卡片式布局，清晰的功能分区
  - 可滚动的内容区域，支持长列表
  - 实时进度显示和状态反馈

- **便捷操作**
  - Windows 原生多选文件夹对话框
  - 打开输出文件夹 + 复制路径到剪贴板
  - 双击编辑视频/目录切分设置
  - 日志实时显示

- **高效处理**
  - 多目录并发处理，提升效率
  - 基于 FFmpeg 的单帧精确定位提取
  - 实时进度同步

## 依赖安装

### 1. 安装 Python

**Windows:**
1. 访问 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.8+ 版本
3. 安装时勾选 **"Add Python to PATH"**

### 2. 安装 FFmpeg

**Windows 安装**

**方法1：使用 Chocolatey（推荐）**
```powershell
# 以管理员身份运行 PowerShell
choco install ffmpeg
```

**方法2：手动安装**
1. 访问 [FFmpeg 官方下载页](https://ffmpeg.org/download.html#build-windows)
2. 下载 Windows builds from gyan.dev
3. 解压到 `C:\ffmpeg` 目录
4. 添加环境变量：
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在系统变量中找到 `Path`，点击编辑
   - 添加 `C:\ffmpeg\bin`
5. 重启命令行，验证安装：
   ```cmd
   ffmpeg -version
   ```

## 使用说明

### 单视频切分模式

1. **选择视频** - 点击"选择视频"按钮
2. **设置切分张数** - 输入或点击快捷按钮设置
3. **选择输出目录** - 点击"选择目录"设置输出位置
4. **开始提取** - 点击"开始提取"按钮

### 单目录切分模式

1. **选择目录** - 点击"选择目录"按钮
2. **设置切分模式**
   - 统一设置：所有视频使用相同张数
   - 单独设置：双击视频列表中的项单独设置
3. **开始提取** - 点击"开始提取"按钮

### 多目录并发切分模式

1. **添加目录** - 点击"添加目录"按钮，支持多选文件夹
2. **设置切分参数**
   - 使用"应用到所有目录"快速设置统一值
   - 或双击目录列表单独设置每个目录
3. **并发数设置** - 调整"并发数"控制同时处理的目录数量
4. **开始提取** - 点击"开始提取"按钮

## 文件命名格式

输出文件命名规则：

```
月日时分_三位随机数_四位递增数_宽_高.格式
```

**示例：**
- `03271245_789_0001_1920_1080.jpg`
- `03271245_789_0002_1920_1080.jpg`

**各部分含义：**
- `03271245` → 03月27日 12:45
- `789` → 三位随机数（100-999）
- `0001` → 四位递增序号
- `1920_1080` → 视频分辨率 宽x高
- `.jpg` → 图片格式

## 常见问题

### Q: 提示 "未检测到FFmpeg"
A: FFmpeg 未正确安装或未添加到系统 PATH。请重新安装并确保 `ffmpeg -version` 命令可以正常执行。

### Q: 多目录选择对话框无法多选
A: 此功能需要 Windows 系统，使用系统原生对话框实现。在非 Windows 系统上会自动回退到单选模式。

### Q: DirectorySettingsDialog 报错
A: 此问题已在 v6.0 中修复。如果仍遇到问题，请确保使用最新版本代码。

### Q: 处理速度较慢
A:
- 单帧提取模式比连续提取慢，但实现了精确控制
- 视频分辨率越高处理越慢
- 使用 JPG 格式比 PNG 格式更快
- 多目录并发模式可提升整体处理效率
