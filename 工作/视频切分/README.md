# 🎬 视频帧提取工具 v3.0

基于 FFmpeg 的视频帧提取桌面应用程序，支持按帧率或总数切分视频为图片。

---

## ✨ 功能特性

- 🎨 **现代化UI** - 深色标题栏 + 卡片式布局 + 真实同步进度条
- 📹 **FFmpeg 实时重命名** - 提取时直接生成目标格式文件名
- ⚡ **高效处理** - 单帧提取模式，边提取边重命名
- 🎯 **快捷设置** - 50/100/150/200 一键设置提取张数
- 📂 **便捷操作** - 打开输出文件夹 + 复制路径到剪贴板
- 🔧 **编码修复** - 修复 Windows 控制台编码错误

---

## 🔧 依赖安装

### 1. 安装 Python

**Windows:**
1. 访问 [python.org](https://www.python.org/downloads/)
2. 下载 Python 3.6+ 版本
3. 安装时勾选 **"Add Python to PATH"**

**Mac:**
```bash
brew install python3
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install python3 python3-pip
```

### 2. 安装 FFmpeg

#### Windows 安装

**方法1：使用 Chocolatey（推荐）**
```powershell
# 以管理员身份运行 PowerShell
choco install ffmpeg
```

**方法2：手动安装**
1. 访问 [FFmpeg 官方下载页](https://ffmpeg.org/download.html#build-windows)
2. 下载 Windows builds from gyan.dev（选择 `ffmpeg-release-essentials.7z`）
3. 解压到 `C:\ffmpeg` 目录
4. 添加环境变量：
   - 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   - 在系统变量中找到 `Path`，点击编辑
   - 添加 `C:\ffmpeg\bin`
5. 重启命令行，验证安装：
   ```cmd
   ffmpeg -version
   ```

#### Mac 安装

```bash
brew install ffmpeg
```

验证安装：
```bash
ffmpeg -version
```

#### Linux 安装

**Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install ffmpeg
```

**CentOS/RHEL:**
```bash
sudo yum install epel-release
sudo yum install ffmpeg
```

**Arch Linux:**
```bash
sudo pacman -S ffmpeg
```

验证安装：
```bash
ffmpeg -version
```

## 📁 文件命名格式

输出文件命名规则：

```
月日时分_三位随机数_四位递增数_宽_高.格式
```

**示例：**
- `03271245_789_0001_1920_1080.jpg`
- `03271245_789_0002_1920_1080.jpg`
- `03271245_789_0003_1920_1080.jpg`

**各部分含义：**
- `03271245` → 03月27日 12:45
- `789` → 三位随机数（100-999）
- `0001` → 四位递增序号
- `1920_1080` → 视频分辨率 宽x高
- `.jpg` → 图片格式

## 🎯 使用说明

### 快速开始

1. **选择视频** - 点击"选择视频"按钮选择要处理的视频文件
2. **设置模式** - 默认使用"按总数提取"模式，默认提取100张
3. **快捷设置** - 点击 50/100/150/200 按钮快速设置提取张数
4. **选择输出目录** - 点击"选择目录"设置输出位置
5. **开始提取** - 点击"🚀 开始提取"按钮
6. **查看结果** - 处理完成后可点击"打开输出文件夹"或"复制输出路径"

### 切分模式说明

**按帧率提取**
- 每秒提取指定帧数
- 适合需要固定时间间隔的场景
- 例：FPS=1 表示每秒提取1帧

**按总数提取（推荐）**
- 从视频中均匀提取N张图片
- 适合快速预览视频内容
- 快捷按钮：50/100/150/200 张

## ⚡ 性能说明

本工具使用 **FFmpeg + 实时重命名** 方案：

1. **单帧提取模式** - 使用 `-ss` 和 `-vframes 1` 精确定位提取
2. **边提取边重命名** - 每提取一帧立即重命名为目标格式
3. **真实进度同步** - 基于已处理帧数计算并显示实时进度
4. **格式优化** - JPG使用 `-q:v 2`，PNG使用 `-compression_level 3`

## 🐛 常见问题

### Q: 提示 "未检测到FFmpeg"
A: FFmpeg 未正确安装或未添加到系统 PATH。请重新安装并确保 `ffmpeg -version` 命令可以正常执行。

### Q: 控制台出现 UnicodeDecodeError
A: 此问题已在 v3.0 中修复。如果仍遇到问题，请确保使用最新版本代码。

### Q: 输出目录不存在时报错
A: 程序会自动创建输出目录，如果仍报错，请检查是否有写入权限。

### Q: 处理速度较慢
A:
- 单帧提取模式比连续提取慢，但实现了实时重命名
- 视频分辨率越高处理越慢
- 使用 JPG 格式比 PNG 格式更快
- 首次提取需要等待 FFmpeg 定位到指定时间点

### Q: 文件名中没有分辨率信息
A: 请确保视频文件包含有效的视频流，程序会尝试从视频中读取分辨率。
