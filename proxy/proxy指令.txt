一行命令（走代理 + 强制官方源）
    HTTP_PROXY=http://10.30.44.154:1111 HTTPS_PROXY=http://10.30.44.154:1111 pip install nvidia-cudnn-cu11==8.9.7.29 -i https://pypi.org/simple --extra-index-url https://pypi.nvidia.com


HTTP_PROXY=你的代理地址 HTTPS_PROXY=你的代理地址 pip install 包名==版本号 -i https://pypi.org/simple --extra-index-url https://pypi.nvidia.com

-i https://pypi.org/simple：强制主源走官方 PyPI，避免默认的清华源（清华不镜像 NVIDIA 包）。
--extra-index-url https://pypi.nvidia.com：添加 NVIDIA 官方仓库，大部分 nvidia-* 包都在这里。
代理部分：根据你习惯用环境变量或 --proxy 都可以，效果一样。

这样写基本能覆盖所有 NVIDIA pip 包的安装需求。如果以后遇到其他包（非 NVIDIA），通常只用代理 + -i https://pypi.org/simple 就够了，不需要额外加 NVIDIA 的 url。
